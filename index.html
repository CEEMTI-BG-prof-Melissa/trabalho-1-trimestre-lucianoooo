Biblioteca digital
Funcionais Não Funcionais Inversos

Funcionais

  • O sistema deve garantir que os alunos consigam visualizar quais livros estão disponiveis.

  • O sistema deve reservar exemplares com antecedência.

  • O sistema deve verificar quais livros estão disponiveis antes mesmo de ir á biblioteca.

  • O sistema deve reservar um exemplar antecipadamente.

  • O sistema deve garantir que consiga pegar o livro que precisa sem correr o risco de chegar tarde demais.

  • O sistema deve permitir que os usuários acessassem atividades complementares aos livros recomendados.

  • O sistema deve verificar quais livros estão disponíveis e quais já foram emprestados.

  • O sistema deve enviar notificações aos usuários sobre a data de devolução dos livros.

  • O sistema deve mostrar recomendações aos usuários com base nos livros lidos anteriores.

  • O sistema deve manter um controle de empréstimos e devoluções.

  • O sistema pode contar com opções de livros digitais.

  • O usuário deve reservar seus exemplos com 2 dias de antecedência.

  • O usuário deve solicitar acesso ao sistema digital da biblioteca.

  • O sistema deve ser de fácil acesso.

Não Funcionais

  • Apenas integrantes da escola(alunos e professores) podem ter acesso ao sistema.

  • O prazo máximo de posse dos livros da biblioteca será de 20 dias.

  • Interessados em recursos digitais e precisam de um sistema que os incentiva à leitura de uma maneira mais acessível.

  • O progresso deve conter um sistema para discussão dos livros lidos, somente para os alunos.

Inversos

  • O sistema não deve permitir que os usuários mantenham posse dos máteriais por tempo indeterminado.

  • O sistema não deve manter a lista dos máteriais mais utilizados privada.

  • O sistema não deve conceder a permissão para pessoas externas.
